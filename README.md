# RadTempCF - Radiology Templates Cloudflare

A comprehensive collection of radiology report templates designed for healthcare professionals, featuring standardized reporting formats, RADS classifications, and AI integration support.

## Features

- **Standardized Templates**: Pre-built templates for various imaging modalities
- **RADS Classifications**: Support for LI-RADS, TI-RADS, PI-RADS, and more
- **JSON Schema Validation**: Ensure template consistency with built-in validators
- **CI/CD Pipelines**: Automated testing and deployment
- **Cloudflare Deployment**: Fast, secure hosting on Cloudflare Pages

## Supported Modalities

- X-Ray
- CT (Computed Tomography)
- MRI (Magnetic Resonance Imaging)
- Ultrasound
- Nuclear Medicine
- Fluoroscopy
- Mammography
- Angiography

## Quick Start

```bash
# Clone the repository
git clone https://github.com/devfinprojects/Radtempcf.git

# Install dependencies
npm install

# Validate templates
npm run validate

# Build the project
npm run build

# Deploy to Cloudflare
npm run deploy
```
